# react-native-incall-manager-light

Incall manager for ReactNative

## Installation

```sh
npm install react-native-incall-manager-light
```

## Usage

```js
import { multiply } from 'react-native-incall-manager-light';

// ...

const result = await multiply(3, 7);
```
